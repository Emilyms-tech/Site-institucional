# Site Institucional IFB

Este projeto é um site institucional sobre o curso Técnico Integrado em Informática do Instituto Federal de Brasília (IFB).

A ideia do projeto foi praticar desenvolvimento web e organizar informações sobre o curso em um site simples e funcional.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## O que o site possui

- Página inicial apresentando o curso
- Estrutura básica de navegação entre páginas
- Layout simples organizado com CSS

## Como visualizar o projeto

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador

Ou acesse diretamente pelo GitHub Pages:

(https://emilyms-tech.github.io/Site-institucional/)

## O que aprendi com esse projeto

- Estruturar páginas usando HTML
- Organizar o layout com CSS
- Adicionar interações simples com JavaScript
